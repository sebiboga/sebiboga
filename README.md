## 🚀 Major Projects & Contributions

### 🌟 peViitor.ro - Job Aggregation Engine

**Open-source job search platform aggregating all career opportunities in Romania**

- **Tech Stack:** PHP, Apache Solr, Apache JMeter, Docker, GitHub Actions
- **Scale:** Daily updates from 200+ companies and national employment agency (ANOFM)
- **Impact:** Thousands of job seekers find opportunities through transparent, centralized platform
- **Architecture:** Decentralized scraper network, real-time indexing, free access for all users

**Key Features:**
- Real-time job aggregation with daily updates
- Company-specific scrapers maintained by volunteer community
- No user registration required (privacy-first approach)
- Mobile-responsive search interface
- Free and open-source platform

**Repository:** [peviitor-ro](https://github.com/peviitor-ro)

### 💪 humans-txt Initiative

**Promoting the initiative that "WE ARE PEOPLE NOT MACHINES"**

- Advocacy project for human-centric technology development
- Supporting humanstxt.org movement
- Building awareness around digital humanity

**Repository:** [humans-txt](https://github.com/sebiboga/humans-txt) (156 ⭐)

### 🗺️ Romanian Data as JSON

**Open datasets for Romanian administrative divisions**

- Complete list of Romanian cities/localities in JSON format
- Community-driven data maintenance
- Used by multiple projects for geographic data

**Repository:** [peviitor-ro/orase](https://github.com/peviitor-ro/orase)
